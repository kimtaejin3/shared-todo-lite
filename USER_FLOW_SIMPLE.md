## 🔄 주요 사용자 여정

### A. 개인 일정 관리
```
시작
  ↓
My Schedule 페이지 진입
  ↓
원하는 날짜 선택 (Mon-Sun)
  ↓
해당 날짜의 일정 타임라인 확인
  ↓
현재 진행중인 일정 확인 (파란색 강조)
  ↓
끝
```

### B. 일정 추가
```
시작
  ↓
+ 버튼 클릭
  ↓
Add Schedule 페이지 진입
  ↓
제목, 시간 입력 (필수)
  ↓
설명, 참여자 입력 (선택)
  ↓
Create 버튼 클릭
  ↓
My Schedule로 리다이렉트
  ↓
끝
```

### C. 팀원 초대
```
시작
  ↓
Team Workspace 진입
  ↓
Invite 버튼 클릭
  ↓
초대 링크 모달 열림
  ↓
Copy Link 클릭
  ↓
팀원에게 공유
  ↓
팀원이 링크로 가입 신청
  ↓
Pending Invites에서 Accept
  ↓
팀에 멤버 추가
  ↓
끝
```

### D. 할 일 참여
```
시작
  ↓
Team Workspace 진입
  ↓
멤버 선택
  ↓
멤버의 할 일 목록 확인
  ↓
참여하고 싶은 할 일 발견
  ↓
Join 버튼 클릭
  ↓
참여 요청 전송
  ↓
멤버가 Join Requests에서 확인
  ↓
멤버가 Accept 클릭
  ↓
할 일에 참여자로 추가
  ↓
끝
```

### E. 참여 요청 처리
```
시작
  ↓
Team Workspace 진입
  ↓
🟠 Join requests 배지 확인
  ↓
Join requests 섹션 펼치기
  ↓
요청 내용 확인
  - 요청자: Mike Chen
  - 할 일: "Update dashboard UI"
  ↓
결정
  ├─ Accept → 참여자에 추가
  └─ Decline → 요청 거절
  ↓
끝
```

## 📊 상태 및 색상 가이드

### 일정 상태
- 🔵 **현재 진행중** - 파란색 배경 (bg-blue-500)
- ⚪ **다른 시간** - 흰색 배경 (bg-white)

### 할 일 상태
- ✅ **Completed** - 초록색 테두리 (border-green-500)
- 🔵 **In Progress** - 파란색 테두리 (border-blue-500)
- ⚪ **Pending** - 회색 테두리 (border-gray-300)

### 알림 배지
- 🔴 **워크스페이스 초대** - 빨간색 (bg-red-500)
- 🟠 **할 일 참여 요청** - 오렌지색 (bg-orange-500)

### 버튼
- **주요 액션** - 파란색 (bg-blue-500)
- **보조 액션** - 연한 파란색 (bg-blue-50)
- **참여 액션** - 연한 오렌지 (bg-orange-50)
